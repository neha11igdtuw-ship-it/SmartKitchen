E‑commerce profile & insights

Like Codolio merges activity from different coding profiles into one summary, SmartKitchen merges what you log from each shopping source (Zepto, Flipkart, Amazon, …) into category and spend insights for your kitchen.

How it works

Linked sources — Add each app you use (Zepto, Flipkart Grocery, Amazon Fresh, …). Optionally save a bookmark URL to your orders page for your own reference.

Log items — Record what you bought (name, category, date, amount). We merge all entries into one dashboard: totals by food category and by store, same idea as combining Codeforces + LinkedIn stats on a portfolio — but for groceries.

Privacy & data access: Zepto, Flipkart, and Amazon do not offer public “paste a link and we pull your orders” APIs to third-party apps. We never scrape those sites. Only data you save here is stored — under your account — and used for this summary.

Import from order screenshot

Screenshot your My orders or order-detail screen from Zepto, Blinkit, Flipkart, Amazon, etc. We use Gemini vision (same API as AI Insights) to read text and tables from the image — comparable to OCR plus layout understanding. Confirm the lines, set store and time, then save. The image is sent only to Google’s API for parsing; SmartKitchen does not keep the image file.
